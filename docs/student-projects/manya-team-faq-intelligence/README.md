## Objective
This proposal outlines Phase 1 improvements for the SAMAGAMA FAQ system by adding intelligence-driven FAQ discovery, priority handling, similar-question suggestions, and admin analytics foundations.

## Proposed Phase 1 Features

### 1. Priority Heatmap on FAQs
FAQs are ranked by urgency using signals such as duplicate questions, frequency of searches, and unresolved feedback. Critical FAQs float to the top with visual priority indicators.

### 2. Smart Similar Questions
When a user opens an FAQ, the system recommends 3–4 semantically similar questions to help students resolve related doubts in one flow.

### 3. Live Search with Intent Detection
The FAQ search experience supports real-time matching, typo tolerance, and basic intent detection for queries such as “when”, “who”, and “how”.

### 4. Feedback and Unresolved Tracker
Each FAQ can collect helpful/not-helpful feedback. Negative feedback contributes to urgency scoring and helps admins identify content that needs improvement.
